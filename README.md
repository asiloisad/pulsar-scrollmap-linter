# scrollmap-linter

Linter message markers for [scrollmap](https://github.com/asiloisad/pulsar-scrollmap). Shows errors, warnings, and info messages. Requires [linter](https://web.pulsar-edit.dev/packages/linter) or [linter-bundle](https://web.pulsar-edit.dev/packages/linter-bundle) package.

## Installation

To install `scrollmap-linter` search for [scrollmap-linter](https://web.pulsar-edit.dev/packages/scrollmap-linter) in the Install pane of the Pulsar settings or run `ppm install scrollmap-linter`. Alternatively, you can run `ppm install asiloisad/pulsar-scrollmap-linter` to install a package directly from the GitHub repository.
